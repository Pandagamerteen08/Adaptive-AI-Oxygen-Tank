## Current Features

- Advanced oxygen tank with AI voice assistance
- Multiple voice variants and personality modes
- Customizable audio packs
- Emergency oxygen reserve system
- Custom Unity AssetBundle model support
- Configurable alerts and thresholds
- Preset system for user configurations

## Project Scope & Expansion

The project is designed to expand beyond a single oxygen tank to include:
- Additional equipment types
- Shared AI core system
- Modular architecture for future content
- Support for community-created equipment mods
