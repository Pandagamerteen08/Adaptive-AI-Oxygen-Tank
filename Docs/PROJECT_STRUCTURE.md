## File Descriptions

### Configuration Files

#### `.github/ISSUE_TEMPLATE/`
- **bug_report.md** - Template for reporting bugs with environment details
- **feature_request.md** - Template for suggesting new features
- **config.yml** - GitHub Issues configuration and links

### Audio Files

#### `Audio/`
- **VoiceLines/** - Organized voice audio files (OGG and WAV)
- **CustomVoicePacks/** - User-created and community voice packs
- **Audio.pak** - Packed audio archive loaded at runtime

### Assets

#### `Assets/`
- **Models/** - Custom tank model bundles (Unity AssetBundles)

### Documentation

#### `Docs/`
- **CHANGELOG.md** - Complete version history and changes
- **METADATA.md** - Project metadata and overview
- **PROJECT_STRUCTURE.md** - This file, directory structure guide
- **ConfigurationReference.md** - All configuration options
- **VoicePackGuide.md** - How to create custom voice packs
- **ModelGuide.md** - How to create custom tank models
- **Credits.md** - Attribution and contributors

## Configuration Files

### User Configuration
- **config/AdaptiveAIOxygenTank.cfg** - Main configuration file for users
  - Tank capacity
  - Voice settings
  - Personality modes
  - Alert toggles
  - Reserve settings

### BepInEx Configuration
- **BepInEx/config/com.bipnx.subnautica.adaptiveaioxygentank.cfg** - Auto-generated by BepInEx

## Build Artifacts

- **AdaptiveAIOxygenTank.dll** - Compiled BepInEx plugin
- **AdaptiveAIOxygenTank.pdb** - Debug symbols for troubleshooting

## Important Notes

1. **Do Not Edit** `BepInEx/config/com.bipnx.subnautica.adaptiveaioxygentank.cfg` directly - use the user config instead.
2. **Audio Management** - All voice audio is packaged in `Audio.pak` for releases.
3. **Custom Assets** - Models must be packaged as Unity AssetBundles.
4. **Versioning** - Follows semantic versioning (MAJOR.MINOR.PATCH[-TAG]).
