# IMPORTANT README

Do not edit `com.bipnx.subnautica.adaptiveaioxygentank.cfg`.

That CFG file is the BepInEx-managed plugin file. It exists so 

BepInEx can register and maintain the mod. Editing it by hand 

can stop the mod from loading or cause settings to reset.

Edit `AdaptiveAIOxygenTank.cfg` instead. That is the user-facing 

configuration file for items, recipe values, AI behavior, voice settings, 

audio volume, oxygen capacity, and gameplay tuning.

If you are installing this mod for someone else, leave both CFG files in 

the `config` folder and only change `AdaptiveAIOxygenTank.cfg` when you need to customize the mod.
