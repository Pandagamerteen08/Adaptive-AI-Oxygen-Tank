# Voice Line Audio Files

Put the custom AI voice clips in this folder.

Supported formats:

- `.ogg` recommended
- `.wav` supported for testing

Recommended voice export:

- OGG Vorbis
- 48 kHz
- Mono
- Keep clips under 4 seconds when possible

Default filenames used by the mod:

- `startup_systems_online.ogg`
- `startup_adaptive_module.ogg`
- `startup_life_support.ogg`
