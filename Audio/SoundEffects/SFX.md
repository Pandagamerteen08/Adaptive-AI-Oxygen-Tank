# Sound Effects

Use this folder for non-voice sounds.

Recommended format:

- OGG Vorbis
- 48 kHz
- Mono
- Under 2 seconds for UI/equipment sounds

Search terms:

- `sci fi ui beep`
- `futuristic equipment startup`
- `oxygen warning alarm`
- `life support alert`
- `air tank hiss`
- `underwater equipment beep`
- `soft computer confirmation beep`
- `futuristic shutdown sound`

Possible future filenames:

- `tank_equip.ogg`
- `tank_unequip.ogg`
- `efficiency_shift.ogg`
- `critical_warning.ogg`
- `surface_standby.ogg`

The current mod only plays AI voice clips from `Audio/VoiceLines`.
